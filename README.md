# Agenda-2030-App-goal-12
Applicazione android in sviluppo per il goal 12 dell'agenda 2030 dell'unione Europea

# Stato del progetto: sviluppo bloccato
A causa delle restrizioni della pandemia iniziata nel 2020, l'istituto Benedetto Castelli di Brescia ha deciso di annullare i progetti dell'agenda 2030 in sviluppo.
